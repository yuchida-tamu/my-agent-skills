# English Coach Plugin

Passive English language coaching that runs in every Cowork session.

## What it does

After answering your question or completing your task, Claude appends brief English corrections and suggestions at the end of the response. Covers four areas:

- **Grammar** — fixes errors in tense, articles, plurals, etc.
- **Natural phrasing** — rewrites awkward sentences into native-sounding alternatives
- **Vocabulary** — suggests richer word choices
- **Business English** — polishes professional communication

## Usage

No commands needed — the skill runs automatically on every message. Corrections appear at the end of each response. If your English is perfect, nothing is appended.

## Configuration

No setup required. Install the plugin and it works immediately.
